# XOR運算[python2]

https://en.wikipedia.org/wiki/XOR_cipher

```
>>> a=0b01010111011010010110101101101001
>>> k=0b11110011111100111111001111110011
>>> c=bin(a^k)
>>> print c
0b10100100100110101001100010011010
>>> d=bin(a^a^k)
>>> print d
0b11110011111100111111001111110011
```
